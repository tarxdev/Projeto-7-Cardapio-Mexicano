Sobre o Projeto

Este projeto é um cardápio digital moderno e visualmente atraente para o restaurante "Tierra Mexicana". Foi desenvolvido para oferecer uma experiência de usuário imersiva e interativa, tornando a navegação pelos pratos uma prévia deliciosa da culinária oferecida.

  • ✔️ Cards interativos com efeito "flip".
  • ✔️ Layout de página única com navegação suave.
  • ✔️ Animações dinâmicas durante a rolagem.
  • ✔️ Design 100% responsivo para celulares e tablets.
