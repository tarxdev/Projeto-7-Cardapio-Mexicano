Sobre o Projeto

Este projeto é um cardápio digital moderno e visualmente atraente para o restaurante "Tierra Mexicana". Foi desenvolvido para oferecer uma experiência de usuário imersiva e interativa, tornando a navegação pelos pratos uma prévia deliciosa da culinária oferecida, com foco em uma apresentação que desperta o apetite. A estrutura foi pensada para ser intuitiva e completa, guiando o cliente por todas as seções do menu de forma fluida e agradável.

  • ✔️ Cards interativos com efeito "flip".
  • ✔️ Layout de página única com navegação suave.
  • ✔️ Animações dinâmicas durante a rolagem.
  • ✔️ Slideshow com pratos em destaque.
  • ✔️ Design 100% responsivo para celulares e tablets.
  • ✔️ Seção com depoimentos de clientes.
